Спочатку обери шлях підбору

Після вибору система або покаже картки сценаріїв для ручного запуску, або запропонує розумне AI-визначення на основі твого текстового опису.

Ручний режим — коли ти вже знаєш, який тип ПК тобі потрібен.
AI-режим — коли хочеш описати задачі простими словами й отримати підказку автоматично.

Ручний вибір типу ПК

Обери сценарій використання, і система відкриє відповідний конфігуратор з потрібними параметрами.

Ігровий ПК

Потужна система для сучасних ігор, потрібного FPS і конкретних графічних налаштувань.
Офісний ПК

Стабільний ПК для документів, браузера, багатозадачності, пошти та робочих застосунків.
ПК для навчання

Універсальна система для навчання, Zoom, програмування, презентацій і щоденного використання.
ПК для монтажу / 3D

Продуктивна конфігурація для монтажу відео, графіки, 3D-моделювання та рендерингу.

Локальна ML-модель

Опиши задачі простими словами

Просто напиши, для чого потрібен комп'ютер: ігри, бухгалтерія, навчання, програмування, монтаж відео, Blender, Photoshop, AutoCAD або будь-які інші задачі.

  • Працює локально без зовнішніх API.
